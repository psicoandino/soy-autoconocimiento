# Soy Autoconocimiento Mockup (Static Proposal)

This folder contains a static HTML/CSS/JS mockup you can publish on GitHub Pages.

## Files

- `index.html` - single-page proposal mockup
- `styles.css` - styling
- `script.js` - mock interactions (login toggle, likes, etc.)

## Reusing an Existing Website (Text + Look)

Put the existing site you want to reference into `reference_site/` (HTML/CSS/JS + images).

Common ways to do that:

1. If you have a folder on your computer:
   - Copy the whole folder into `reference_site/`.

2. If you have a `.zip`:
   - Unzip it, then move the unzipped folder into `reference_site/`.

Once the files are here, I can:

- extract text content and rebuild it into this mockup
- match colors, typography, spacing, and components
- reuse images (or set up an `assets/` folder)

Note: make sure you have permission to reuse the original design/assets.
